Инструкция

  1. Загрузите ваш JSON файл, созданный согласно заданию.
  2. Нажмите кнопку "Проверить".
  3. Система сравнит ваш файл с эталонным и покажет результат.
  4. Если файлы идентичны, вы получите секретный код для подтверждения выполнения задания.
  5. Если есть различия, система покажет их и даст рекомендации по исправлению.
